# FlyerScanCT
Computer aided detection of lung nodules in CT scans [1-3].

# Test Data

A test case may be found on The Cancer Imaging Archive (TCIA) here:
https://wiki.cancerimagingarchive.net/pages/viewpage.action?pageId=1966254

We used LIDC-IDRI-0003. Download the folder of .dcm files for this case.
